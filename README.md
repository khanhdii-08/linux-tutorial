# Linux Command Tutorial

> Tài liệu đào tạo Linux dành cho người mới bắt đầu  
> Phiên bản 1.0 | Dành cho Ubuntu 22.04 / Debian

---

## 📖 Giới thiệu

Tài liệu này cung cấp kiến thức nền tảng về các lệnh Linux thông dụng – từ điều hướng thư mục, quản lý file, phân quyền, công cụ hệ thống, đến giám sát tài nguyên. Mỗi lệnh đều có giải thích, ví dụ thực tế, lỗi thường gặp và mẹo sử dụng, giúp bạn nhanh chóng làm quen và thực hành hiệu quả.

---

## 📂 Cấu trúc nội dung

| Chương | Nội dung chính |
|--------|----------------|
| 1 | Điều hướng thư mục: `ls`, `cd`, `clear` |
| 2 | Quản lý thư mục và file: `mkdir`, `touch`, `vi` |
| 3 | Đọc và tìm kiếm nội dung: `cat`, `echo`, `tail`, `grep` |
| 4 | Sao chép và xóa: `cp`, `mv`, `rm`, `rmdir` |
| 5 | Quyền truy cập: `sudo`, `chmod`, `chown` |
| 6 | Công cụ hệ thống: `man`, `wget`, `apt` |
| 7 | Tiến trình và mạng: `kill`, `ping`, `uname`, `passwd` |
| 8 | Theo dõi tài nguyên: `top`, `df`, `free` |
| Phụ lục | Cheat Sheet, các cặp lệnh dễ nhầm, workflow thực tế, mini project, 30 bài tập có đáp án |

---

## 🚀 Hướng dẫn sử dụng

1. **Đọc tuần tự** từ Chương 1 đến Chương 8 để nắm vững kiến thức.
2. **Tra cứu nhanh** bằng **Cheat Sheet** ở cuối tài liệu.
3. **Thực hành** các bài tập cuối mỗi chương và 30 bài tập tổng hợp (có đáp án).
4. **Áp dụng** vào workflow thực tế và mini project để ghi nhớ sâu hơn.

---

## 📌 Yêu cầu hệ thống

- Hệ điều hành: Ubuntu / Debian (hoặc bất kỳ bản phân phối Linux nào)
- Terminal và quyền `sudo` cho các lệnh quản trị
- Kết nối Internet (để cài đặt gói và tải file bằng `wget`)
